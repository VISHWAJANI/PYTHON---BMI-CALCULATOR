# BMI CALCULATOR
--> The project is made using Python.


## Features
--> It takes input for weight in kg and height in metre.
--> Calculates BMI
--> Indentifies BMI categories:
 -Underweight : BMI < 18.5
 -Normal : BMI = 18.5 - 24.9
 -Overweight : BMI= 24.9 - 29.9
 -Obese I: BMI = 30 - 34.9
 -Obese II : BMI = 34.9 - 39.9
 -Obese III : BMI > 40

   
## Language Used
--> Python

## How to Run

1. Download or clone this repository.
2. Open the project in Python IDLE or any Python IDE.
3. Run the program.
4. Enter your height and weight when prompted.
